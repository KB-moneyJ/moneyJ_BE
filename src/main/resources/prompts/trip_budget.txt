여행 국가: %s, 여행 도시: %s, 숙박 기간(박): %d, 여행 기간(일): %d, 출발일: %s, 도착일: %s

이 여행의 예상 경비를 JSON 형식으로 반환해.

필드는 flightCost, accommodationCost, foodCost, totalCost 이어야 한다.

flightCost 조건:
- 출발지 인천 국제공항(ICN)을 기준으로 여행 도시 대표 공항 간 왕복 항공권 예상 평균 비용
- 반드시 도시별로 다른 값을 사용해야 하며, GPT가 학습한 과거 항공권 데이터를 참고
- (이코노미 클래스, 저가항공 평균가, 성수기 제외, 원 단위 정수)

accommodationCost 조건:
- 숙박 기간(박) x 여행 도시의 3성급 호텔 1박 평균가
- 도시가 다르면 반드시 값이 달라져야 하며, GPT가 학습한 과거 해당 여행 도시의 호텔 단가 데이터를 참고
- 여행 기간이 아닌 숙박 기간으로 계산

foodCost 조건:
- 여행 도시의 평균 물가를 반영
- 기본 식비 = 여행 기간(일) x 3 x 도시별 한 끼 평균가 x 환율
- 기타 비용 = 여행 기간(일) x (간식 + 음료 + 주류)비용
- 최종 foodCost = 기본 식비 + 기타 비용

totalCost 조건:
- totalCost 는 flightCost + accommodationCost + foodCost + 의 합산 값 (원 단위, 정수 숫자만)

주의 :
- 반드시 JSON 형식으로 출력하고, 다른 설명 문장은 포함하지 말 것
